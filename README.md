# DVWA Vulnerability Assessment Lab

## Overview

This project contains practical vulnerability testing performed using **Damn Vulnerable Web Application (DVWA)**. The objective of this lab is to understand common web application vulnerabilities, demonstrate exploitation techniques in a controlled environment, and identify proper security mitigation methods.

DVWA provides intentionally vulnerable web pages that allow security students to practice penetration testing techniques safely.

---

## Environment Setup

### Tools Used

- **DVWA (Damn Vulnerable Web Application)**
- **XAMPP (Apache + MySQL)**
- **PHP**
- **MySQL Database**
- **Web Browser**
